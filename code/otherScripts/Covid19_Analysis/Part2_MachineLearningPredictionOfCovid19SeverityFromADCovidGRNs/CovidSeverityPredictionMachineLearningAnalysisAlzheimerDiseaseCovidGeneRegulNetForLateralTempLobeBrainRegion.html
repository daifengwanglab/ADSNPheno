Here, we will perform Covid-19 Severity Prediction Machine Learning Analysis for Alzheimer's Disease-Covid Gene Regulatory Network for the Lateral Temporal Lobe. We use 4-Fold Cross Validation (so each fold has 25 patients in it) and Support Vector Machine with a Linear Kernel.

Please note that this demo below is for the Entire (Original) Lateral Temporal Lobe and provides some of the results used in Figure 6 for the Lateral Temporal Lobe AD-Covid GRN and Machine Learning Model for predicting Covid-19 Severity (Intensive Care Unit (ICU) versus Non-Intensive Care Unit (Non-ICU)). Here, Severe Covid-19 patients are considered as those in the ICU.¶
